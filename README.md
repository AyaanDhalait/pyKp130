# pyKp130

**pyKp130** is a programmable virtual science and engineering laboratory built entirely in Python.

It allows you to define experiments as code, simulate them over time, and collect reproducible numerical data — similar to how real scientific instruments work.

---

## What pyKp130 Does

- Runs scientific simulations using model time
- Supports multiple experiment types (physics, biology, systems)
- Logs numerical data automatically
- Exports results as CSV files
- Uses a clean experiment → simulation → data pipeline

This project focuses on **correct simulation structure**, not graphics.
